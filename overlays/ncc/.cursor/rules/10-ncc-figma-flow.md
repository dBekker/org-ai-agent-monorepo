# NCC Figma → Code Rule

For visual tasks, follow this sequence:

1. read `ai_docs/standards/02-figma-to-code.md`
2. identify the target route and section
3. reuse existing UI primitives first
4. build or refine the section
5. wire data only after the structure is correct
6. verify at `375 / 768 / 1280 / 1536`

Pixel-perfect includes:
- spacing
- typography
- wrapping
- CTA sizing
- decorative asset behavior
- no accidental overflow
